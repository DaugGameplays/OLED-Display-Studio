OLED Studio – Credits

OLED Studio ist ein modularer, skalierbarer Editor für Pixelgrafiken auf OLED‑Displays. Entwickelt für Klarheit, Geschwindigkeit und maximale Kontrolle über jedes einzelne Pixel.

OLED Studio is a modular, scalable editor for pixel graphics on OLED displays. Designed for clarity, speed, and full control over every single pixel.

Werkzeuge & Bedienung

Tools & Usage

Pixel setzen

  • Linksklick: Pixel setzen
  • Rechtsklick: Pixel löschen
  • Gedrückt halten: Linien zeichnen

Stempel‑Werkzeug

  • Setzt gespeicherte Muster wie ein Pinsel

Verschieben

  • Shift Up/Down/Left/Right: verschiebt das gesamte Raster

Spiegeln & Rotieren

  • Mirror H: horizontal spiegeln
  • Mirror V: vertikal spiegeln
  • Rotate: 90° drehen

Import & Export

  • HEX, BIN, C‑Array Export
  • Automatische Import‑Erkennung

Drawing Pixels

  • Left click: set pixel
  • Right click: erase pixel
  • Hold mouse: draw continuous lines

Stamp Tool

  • Places saved patterns like a brush

Shifting

  • Shift Up/Down/Left/Right: moves the entire grid

Mirror & Rotate

  • Mirror H: flip horizontally
  • Mirror V: flip vertically
  • Rotate: rotate 90° clockwise

Import & Export

  • HEX, BIN, C‑array export
  • Automatic import detection
